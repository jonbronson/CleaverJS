Mesh

new Mesh()

Creates a new Mesh object

Methods

buildAdjacency()

Build adjacency information so neighbor queries can be made. This includes generating edges, and storing incident faces and edges.

createFace(v1, v2, v3, material) → {Triangle}

Creates a new face for the mesh, using the given vertices. Any vertex not already in the mesh will be added to the vertex list.

Parameters:
Name Type Description
v1 Vertex First vertex of the face.
v2 Vertex Second vertex of the face.
v3 Vertex First vertex of the face.
material number The material of the face.
Returns:
Triangle The newly created face.
