Rect

Creates a new rectangle object

Parameters:
Name Type Description
left number The left x coordinate of the rectangle.
bottom number The bottom y coordinate of the rectangle.
right number The right x coordinate of the rectangle.
top number The top y coordinate of the rectangle.
